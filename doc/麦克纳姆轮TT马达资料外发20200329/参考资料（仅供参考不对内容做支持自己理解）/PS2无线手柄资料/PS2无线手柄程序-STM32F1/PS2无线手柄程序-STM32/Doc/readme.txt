【软件说明】
 * 软件名称：  X-CTR400 扩展例程 PS2手柄程序 V1.0
 * MDK版本：   V5.23 
 * 固件库版本： V3.5.0
 * 主要功能：    
   - X-CTR400 扩展PS2手柄功能

【硬件说明】
 * 硬件平台：X-CTR400 控制器
 * 处理器：STM32F103RCT6
 * 外部高速晶振：8MHz
 * 总线运行时钟：系统时钟 = SYCCLK = AHB1 = 72MHz
 * 硬件原理接口：请参考开源硬件原理图
